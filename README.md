  Python Learning Journey

This repository documents my personal journey of learning Python from the fundamentals to more advanced topics. It is structured in a progressive way, starting from basic syntax and gradually moving toward object-oriented programming, data handling, APIs, testing, and data visualization.

Each section contains practical exercises and mini-projects designed to strengthen both theoretical understanding and hands-on coding skills. The focus is on writing clean, readable, and structured code while gradually building real-world problem-solving abilities.

The exercises included here are only a representative selection of the full set of problems I worked on during my learning process. Many additional practice exercises, experiments, and variations were completed outside of this repository to reinforce understanding.

The main goal of this repository is to build a strong foundation in Python programming and develop the ability to write clean, structured, and practical code applicable to real-world scenarios.
